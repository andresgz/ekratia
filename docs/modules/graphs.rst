Graphs
======

Ekratia uses `Networkx <http://networkx.github.io/>`_ library to operate with 
graphs. GraphEkratia is a class that extends from  
`DiGraph <http://networkx.github.io/documentation/latest/reference/classes.digraph.html>`_ 
and implements required methods for the Ekratia vote system.
